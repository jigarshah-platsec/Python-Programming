# Heaps: Core Foundation

## Mental Model
A binary tree where the parent is always smaller/larger than its children. Used for finding Top K or Min/Max efficiently.

## Key Patterns
- **Top K Elements:** Using a MinHeap of size K.
- **Merge K Sorted Lists:** Pushing the head of each list into a heap.
- **Two Heaps:** Maintaining a median in a data stream (MaxHeap for lower half, MinHeap for upper half).
