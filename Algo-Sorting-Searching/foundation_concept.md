# Sorting & Searching: Core Foundation

## Mental Model
- **Sorting:** Ordering elements (Merge Sort, Quick Sort, etc.).
- **Searching:** Finding elements efficiently (Binary Search).

## Key Patterns
- **Binary Search:** $O(\log N)$ search on sorted arrays. Can also be used to find a threshold/boundary.
- **Custom Sort:** Sorting by specific criteria (e.g., length, frequency).
