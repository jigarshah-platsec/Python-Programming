# Trees: Core Foundation

## Mental Model
Hierarchical data structure with a root node and child nodes.

## Key Patterns
- **DFS (Depth-First Search):** Pre-order, In-order, Post-order traversal (using Recursion or Stack).
- **BFS (Breadth-First Search):** Level-order traversal (using Queue).
- **Binary Search Tree (BST):** Left child < Parent < Right child.
