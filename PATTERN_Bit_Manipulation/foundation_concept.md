# Pattern: Bit Manipulation

## Mental Model
Manipulating data at the bit level using operators like AND (`&`), OR (`|`), XOR (`^`), and shifts (`<<`, `>>`).

## Key Patterns
- **XOR:** $x \oplus x = 0$. Used for finding the single number in an array of pairs.
- **Checking Powers of 2:** `n & (n - 1) == 0`.
