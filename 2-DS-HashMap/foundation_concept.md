# HashMaps: Core Foundation

## Mental Model
HashMaps (Dictionaries in Python) map keys to values using a hash function, providing $O(1)$ average time complexity for lookups, insertions, and deletions.

## Key Patterns
- **Frequency Counting:** Counting occurrences of elements.
- **Caching / Memoization:** Storing previously computed results.
- **Grouping:** Grouping items by a specific property (e.g., anagrams).
