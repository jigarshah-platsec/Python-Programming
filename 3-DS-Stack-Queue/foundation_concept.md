# Stacks and Queues: Core Foundation

## Mental Model
- **Stack (LIFO):** Last In, First Out. (Like a stack of plates).
- **Queue (FIFO):** First In, First Out. (Like a line at a store).

## Key Patterns
- **Monotonic Stack:** Keeping elements strictly increasing or decreasing (e.g., Next Greater Element).
- **Valid Parentheses:** Using stacks to track open brackets.
- **BFS (Breadth-First Search):** Using a queue to explore level by level.
