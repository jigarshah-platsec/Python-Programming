# Dynamic Programming: Core Foundation

## Mental Model
Breaking down a problem into overlapping subproblems and caching their results to avoid redundant work.

## Key Patterns
- **Top-Down (Memoization):** Recursion + Cache.
- **Bottom-Up (Tabulation):** Iteration + DP Array.
- **1D DP / 2D DP:** e.g., Climbing Stairs, Longest Common Subsequence.
